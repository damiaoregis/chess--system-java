# ♟️ Chess System Java

Projeto em **Java** para implementação de um sistema de xadrez.  
Este repositório foi criado por **Damião Regis de Farias Barbosa** como exercício prático de programação orientada a objetos.

---

## 📂 Estrutura do projeto

- **src/** → código-fonte principal em Java  
- **.idea/** → arquivos de configuração da IDE  
- **.gitignore** → arquivos e pastas ignorados pelo Git  
- **chess-system-java.iml** → arquivo de configuração do projeto  

---

## 🚀 Funcionalidades previstas

- Representação das peças de xadrez e suas regras de movimento  
- Tabuleiro com lógica para validar jogadas  
- Controle de turnos entre jogadores  
- Tratamento de promoção de peões  
- Detecção de situações como xeque e xeque-mate

## 📖 Como executar

1. **Clonar o repositório**
   ```
   git clone https://github.com/damiaoregis/chess-system-java.git
   cd chess-system-java
   ```
2. **Abrir em uma IDE (IntelliJ IDEA ou outra compatível com projetos Java)**
3. **Compilar e executar**
   - Usar o comando da IDE ou:
   ```
   javac src/*.java
   java Main

   ```
